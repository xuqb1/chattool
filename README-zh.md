# chattool
ChatTool 是一个基于Kimi的可多轮对话，可本地执行 Javascript 和 Python 代码的 ai 会话客户端

功能:
  1. 界面语言切换，目前可切换中文和英文
  2. 多轮对
  3. 运行 Javascript 和 Pytho代码，需要本机安装好 node 和 python
  4. 根据我发出的话来进行快速切换，不需要猛爬楼了

用法:
  先全部下载了。
  解压到 chattool 目录
  然后在VSCode或 PyCharm 里加载该目录
  运行 chattool4.py
  单击设置按钮
  在弹出窗口中输入你的 Kimi的 key
  单击保存按钮
  重新运行 chattool4.py
  可以试试会话了
